main.py will look for a .csv file, due to this, keep only one .csv file in the root directory.

Credentials and url for emeger are in the credentials.py file.
See the credentialsExample.py 

Instructions:

To Install:
```
python3 -m venv emerge
source emerge/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
decativate
```

Run script
```
source emerge/bin/activate
python3 main.py
```

After run
```
deactivate
```
